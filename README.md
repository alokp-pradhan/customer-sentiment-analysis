# Customer Sentiment Analysis on Mobile Phone Reviews Using NLP

An end-to-end Natural Language Processing (NLP) and Machine Learning project that analyzes customer reviews of mobile phones and predicts sentiment as Positive, Negative, or Neutral.

The project uses real-world customer feedback data from mobile phone product reviews to understand customer opinions, satisfaction levels, and sentiment patterns.

## Features

* Advanced text preprocessing using NLTK
* Tokenization, stopword removal, and lemmatization
* TF-IDF vectorization with unigrams and bigrams
* Sentiment labeling using VADER Sentiment Analyzer
* Multiple machine learning model training and comparison
* Deep Learning model implementation using TensorFlow/Keras
* Model evaluation using Accuracy, Precision, Recall, and F1-score
* Data visualization using WordClouds and sentiment distribution charts
* Deployment-ready model serialization using Pickle

## Technologies Used

Python, Pandas, NumPy, Scikit-learn, TensorFlow, NLTK, Matplotlib, Seaborn

## Machine Learning Models Used

* Logistic Regression
* Naive Bayes
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)
* Artificial Neural Network (ANN)

## Project Highlights

* Built a complete NLP pipeline from raw mobile phone review text to sentiment prediction
* Compared multiple ML algorithms to identify the best-performing model
* Applied TF-IDF feature engineering for effective text representation
* Achieved strong sentiment classification performance on customer review data
* Created reusable prediction workflow for real-time sentiment analysis

## Real-World Applications

* E-commerce product review analysis
* Customer feedback monitoring
* Mobile brand sentiment tracking
* Consumer satisfaction analysis
* Product quality assessment
